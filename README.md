## Project Description
This project is an audit of an open-source software. It includes:
- Study of open source philosophy
- Linux system understanding
- Shell scripting tasks

---

## Scripts Included

### 1. system_identity.sh
Displays system information like user, kernel, date, and uptime.

### 2. package_inspector.sh
Checks whether a package is installed on the system.

### 3. disk_auditor.sh
Displays disk usage and permissions of important directories.

### 4. log_analyzer.sh
Counts how many times a keyword appears in a log file.

### 5. manifesto_generator.sh
Generates a simple open-source manifesto based on user input.
